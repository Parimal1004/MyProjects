This **Online Examination System** is a full-stack web application designed to facilitate secure and efficient digital testing. Built with a **Node.js** and **Express.js** backend and a dynamic **HTML/CSS/JavaScript** frontend, the system provides a seamless experience for students to take timed exams and receive instant feedback.

### Key Features

  * **Timed Assessments**: Includes an automated 60-second countdown timer for each exam session.
  * **Auto-Submission**: Ensures integrity by automatically submitting the exam once the timer expires.
  * **Instant Evaluation**: Provides immediate results, including scores, percentages, and letter grades upon completion.
  * **Subject-Specific Modules**: Supports multiple question banks for subjects like Java, Python, DSA, and HTML.
  * **Secure Access**: Features a user authentication flow and responsive design for various devices.
